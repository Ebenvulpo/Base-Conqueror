# Base-Conqueror
A real-time strategy game that I made and released on itch.io years ago.

## Dependencies
```
g++
meson
libSDL2-dev
libSDL2-image-dev
libSDL2_mixer-dev
```

## Building
```
git clone ...
cd Base-Conqueror
meson setup build
cd build
meson compile
cp BaseConqueror ../
cd ..
```

## Run
```
./BaseConqueror
```
